1.- Se debe abrir la terminal para desempaquetar y descomprimir el fichero
2.- El archivo se debe ejecutar accediendo a la carpeta Proyecto_Final usando el comando cd con la ruta solicitada
3.- La ejecucion se realiza con ./Proyecto_final.sh
4.- Al aparecer el menu primcipal es necesario ingresar la opcion con letra minuscula a,b,c,d,e
5.- Cada submenu consta de opciones numericas las cuales deben ser colocadas utilizando (1,2,3,4,5) segun sea el caso
6.- Al elegir una opcion que deba ingresar un parametro debe ser exactamente el nombre al cual se quiere acceder
7.- El programa utiliza variables en español por lo que es necesario que su version en linux este configurada en idioma español
8.- Para ingresar a un archivo o directorio es necesario poner la ruta desde la raiz p.ejemplo /etc/passwd
